# App Store Specifications Reference

Quick reference for Apple App Store and Google Play Store requirements.

## Apple App Store

### Metadata Character Limits

| Field | Limit | Editable Without Update |
|-------|-------|------------------------|
| App Name | 30 chars | No |
| Subtitle | 30 chars | No |
| Promotional Text | 170 chars | Yes |
| Description | 4,000 chars | No |
| Keywords | 100 chars | No |
| What's New | 4,000 chars | Yes (with update) |

### Screenshot Requirements

| Device | Size (px) | Min | Max |
|--------|-----------|-----|-----|
| 6.7" Display | 1290 × 2796 | 3 | 10 |
| 5.5" Display | 1242 × 2208 | 3 | 10 |
| iPad 12.9" | 2048 × 2732 | 3 | 10 |

- Format: PNG or JPEG
- Max file size: 8MB per image
- App previews: up to 3 per locale, 15-30 seconds

### App Icon

- 1024 × 1024 px (App Store)
- PNG, no alpha channel
- No rounded corners (system applies mask)

## Google Play Store

### Metadata Character Limits

| Field | Limit |
|-------|-------|
| App Name | 50 chars |
| Short Description | 80 chars |
| Full Description | 4,000 chars |

### Screenshot Requirements

| Type | Min Size | Max Size | Count |
|------|----------|----------|-------|
| Phone | 320px (short) | 3840px (long) | 2-8 |
| 7" Tablet | 320px | 3840px | up to 8 |
| 10" Tablet | 320px | 3840px | up to 8 |

- Aspect ratio: 16:9 or 9:16
- Format: PNG or JPEG (24-bit, no alpha)
- Max file size: 8MB

### Feature Graphic

- 1024 × 500 px (required)
- Used in search results and store listing

### App Icon

- 512 × 512 px
- 32-bit PNG with alpha
