# Roundtable Workflow

思想探索式圓桌討論 — 真實人物視角，求真不求和，用戶互動控制深度。

適用於：哲學、社會、文化、概念性議題 — 沒有標準答案、需要多元視角深挖的問題。

## Prerequisites

- 一個值得探索的議題（哲學、社會、文化、概念性）
- Optional: 用戶指定人物或輪數

## Parameters

| 參數 | 預設 | 說明 |
|------|------|------|
| 人物數 | 3-5 | 可由用戶指定或自動選擇 |
| 最大輪數 | 無上限（用戶控制） | 用戶說「止」結束 |
| 語言 | 跟隨用戶語言 | 中文議題用中文，英文議題用英文 |

## Execution

### Step 1: 解析議題

從用戶輸入中提取核心議題。如果只說「圓桌」未給議題，詢問議題。

### Step 2: 選人 — 提議代表人物

根據議題，選擇 3-5 位**真實歷史或當代人物**作為代表。每位人物需要：
- 姓名（真實人物，非虛構）
- 核心立場（一句話）
- 選擇理由（為什麼此人對此議題有獨特視角）

選人原則：
- 立場必須形成**張力網絡**（非簡單正反方）
- 優先選擇在該領域有**經典著作或知名言論**的人物
- 至少包含一位「意外視角」— 來自議題本身領域之外的人

輸出人物列表，詢問用戶是否同意或要調整。

### Step 3: 開場 — 統一定義

以主持人身份開場，提出**定義性問題**：

> 「在深入探討之前，我們應當如何定義 [議題核心概念]？」

每位參會者依次發言，格式為：

```
【人物名】【行動標籤】：發言內容

**簡言之**：一句話總結
```

行動標籤：`陳述`、`質疑`、`補充`、`反駁`、`修正`、`綜合`

### Step 4: 對話循環

每輪執行以下流程：

**4a. 動態發言輪**
- 根據討論動態決定誰該發言（不是每人固定一次）
- 每人發言必須是對**前面發言的回應**（質疑/補充/反駁），不許自說自話
- 每段發言末尾必須有 `**簡言之**：` 一句話壓縮
- 發言必須**忠於其真實思想體系**，引用/化用其經典著作或知名觀點

**4b. 主持人綜述**
發言結束後，主持人做三件事：
1. 提煉本輪**核心爭議點**（找到最深的裂縫，不面面俱到）
2. 生成 **ASCII 思考框架圖**（矩陣/光譜/因果環/層級樹 — 選最貼合本輪結構的形式）
3. 提出**下一層引導問題**（從核心爭議中生長出來的更深問題）

**4c. 用戶指令**
綜述後展示指令選單：

```
【主持】指令：可 / 止 / 深入此節 / 引入新人物
```

| 指令 | 效果 |
|------|------|
| `可` | 接受下一層問題，繼續推進 |
| `止` | 結束討論，進入總結 |
| `深入此節` | 不推進新問題，繼續圍繞當前爭議點深挖 |
| `引入新人物` | 用戶指定新人物加入，主持人介紹並請其表態 |

### Step 5: 結束 — 生成知識網絡

用戶發出「止」指令後：

1. 主持人做**全局總結**
2. 生成**完整知識網絡** ASCII 圖：標出所有關鍵概念、立場、爭議點及其關係
3. 列出**未解決的開放問題**

### Step 6: 寫入檔案

將討論全貌寫入 Obsidian：

1. 取得時間戳：`date +%Y%m%dT%H%M%S`
2. 寫入：`~/Documents/Project/obsidiant-vault/XYZ-medium/roundtable/{timestamp}--{議題關鍵詞}.md`
3. 檔案結構：

```markdown
---
date: {YYYY-MM-DD}
topic: {議題}
participants: [{人物列表}]
rounds: {輪數}
tags: [roundtable, {相關標籤}]
---

# 圓桌：{議題}

## 參會者
{人物列表與立場}

## 討論紀錄
### 第 N 輪：{引導問題}
#### 發言紀錄
#### 核心爭議
#### 框架圖

## 知識網絡（全局）
## 開放問題
```

4. 向用戶報告檔案路徑

## 主持人行為準則

- **理性之錨**：冷靜客觀，不偏向任何一方
- **挖深不鋪廣**：每輪只追一條最深的裂縫
- **求真 > 和諧**：鼓勵尖銳但有建設性的交鋒，拒絕表面共識
- **元認知**：在綜述中暴露討論的**結構**（假設、前提、推理鏈），不只複述內容

## 參會者行為準則

- 忠於其真實思想體系發言
- 引用/化用其經典著作或知名觀點
- 發言有鋒芒：質疑要見骨，補充要推進
- 每段結尾 `**簡言之**` 一句話壓到極致

## Done

圓桌結束。完整紀錄已存入 Obsidian。
